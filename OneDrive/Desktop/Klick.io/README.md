# Digital Marketing Agency Website

## Overview
This is a responsive and visually appealing website designed for a Digital Marketing Agency. It showcases the agency's services, team, portfolio, and contact information, providing an excellent user experience.

## Features
- Fully responsive design
- Service offerings section
- Portfolio showcase
- About Us section
- Contact form with email integration
- Modern UI/UX design

## Technologies Used
- **HTML**: Structure of the website
- **CSS**: Styling and responsive design
- **JavaScript**: Interactive features and animations
